# GenAI Text Generator Project

## Description
This project demonstrates a Generative AI model that generates human-like text using a pre-trained GPT-2 model.

## Technologies Used
- Python
- Transformers
- Google Colab
- Hugging Face

## How It Works
1. User enters a text prompt
2. The GenAI model processes the input
3. AI generates meaningful text

## How to Run
1. Open notebook in Google Colab
2. Run all cells
3. Enter prompt when asked

## Example
Input: Artificial Intelligence is  
Output: Artificial Intelligence is transforming industries by...
